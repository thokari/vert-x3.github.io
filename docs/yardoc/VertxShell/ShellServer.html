Overview

The shell server.

A shell server is associated with a collection of : the #register_term_server method registers a term server. Term servers life cycle are managed by this server.

When a receives an incoming connection, a instance is created and associated with this connection.

The #create_shell method can be used to create instance for testing purposes.
