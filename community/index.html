Community

Without the community we are nothing.

Vert.x is an independent project and has been created by contributions from many different individuals and organisations.

We have a thriving user and developer community and many exciting new developments to come, so why not be a part of it?

Get Help

User Group

A Google Group to discuss all user issues related to using Vert.x. Please use this group not StackOverflow for posting questions. We don't answer questions on SO.

Developer Group

A group for Vert.x core developers and contributors.

IRC channel

This is our day-to-day office: #vertx on freenode.net. Feel free to chat to the team.

Issues

Please file issues related to the core Vert.x project in Eclipse Bugzilla. File issues related to non-core Vert.x projects in the appropriate GitHub issues.

Vert.x awesome

A curated list of awesome frameworks, libraries or other components for use with Vert.x. Feel free to add yours by submitting a pull request.

Wiki

The Vert.x wiki contains useful information such as a How to contribute guide, the Roadmap and Project Information.
